项目结构
    crud：封装的数据 库增删改查的代码
    models： 模型类
    routers：拆分出来的路由
    schemas：pydantic数据校验
    utils：工具
    config：数据库相关，缓存配置配置相关
模块化路由
    模块化路由就是把每个业务功能的接口拆分到独立的文件里，在统一挂载到主应用中
接口实现流程
    1.模块化路由 -> API 接口规范文档
        定义 APIRouter 实例
        注册路由 include_router()
        参照接口规范文档
    2.定义模型类 -> 数据库表（数据库设计文档）
        参照数据库表
    3.数据库 CRUD -> 在 crud 文件夹里面创建文件，封装操作数据库的方法
        select(模型类)
        add()
        update()
        delete()
    4.路由调用逻辑 -> 在路由处理函数里面 crud 封装好的方法，响应结果
        Depends 注入数据库依赖
        调用逻辑
        响应结果
跨域资源共享CORS
    跨域资源共享（CORS）是一种浏览器安全机制，用于允许运行在一个源（Origin）的Web应用，通过浏览器向另一个源的服务器发起跨域HTTP请求，并在fuwuqi授权的前提下获取资源
    同源的三个条件：协议、域名、端口
    CORS中间件：
        CROS：让后端主动告诉浏览器：这个前端“允许访问”
passlib 密码加密
    安装
        pip install "passlib[bcrypt]==1.7.4"
        # passlib：哈希算法，bcrypt：依赖算法
    创建密码上下文
        pwd_context = CryptContext(schemes=["bcrypt"], deprecated="auto")
        依赖于 CryptContext ，参数一是加密算法（schemes=["bcrypt"]），参数二指代是新旧加密算法，可以自动兼容，自动适应
    加密或校验
        pwd_context.hash(password)  # 传入明文 password 得到加密结果
        pwd_context.verify(plain_password, hashed_password)  # verify方法校验明文和密文是否一致
ORM 联表查询的写法
    select(查询的主题类模型, 字段重命名)join(join的模型类, join的条件)
    select(
        News,
        Favorite.created_at.label("favorite_time"),
        Favorite.id.label("favorite_id")
    ).join(Favorite, Favorite.news_id == News.id)
